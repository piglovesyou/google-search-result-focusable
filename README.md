# Develop

- Run `yarn build`
- In `chrome://extensions/` load "unpacked" of `./build`
- Verify it in Google search result page

# Publish (for me)

- `yarn build`
- Upload `build.zip` to https://chrome.google.com/webstore/developer/dashboard
